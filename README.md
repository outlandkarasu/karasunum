A numerical library for personal use.

## Modules

* `karasunum.linear_algebra`
    * static dimensions vector and matrix
    * LU decomposition
* `karasunum.optimize`
    * Newton method
* `karasunum.differential`
    * auto differential tool kit
* `karasunum.time_series.kalman_filter`
    * Kalnam filter
